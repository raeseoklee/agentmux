# AgentMux 0.1.3 릴리즈 증적

일자: 2026-06-29

## 범위

AgentMux 0.1.3은 공개 릴리즈 준비에 초점을 둔 Windows-only 패치
릴리즈입니다.

- Node, Tauri, Rust workspace, lock 파일의 버전을 0.1.2에서 0.1.3으로
  올렸습니다.
- 공개 README, 보안 정책, 이슈 템플릿, PR 템플릿, 알려진 제한사항 문서를
  정리했습니다.
- 영어와 한국어 릴리즈 정책 문서를 추가했습니다.
- GitHub release workflow의 publish 경로에서 사용하는 action을 commit SHA로
  고정했습니다.

## 게시 전 검증

태그를 푸시하기 전에 다음 로컬 검증을 완료했습니다.

- `npm run version:check -- --tag v0.1.3`
  - Codexus 증적:
    `.codexus/session/verification/verification_20260629_053610_908e9d/verification.json`
- `npm --prefix apps/desktop run build`
  - Codexus 증적:
    `.codexus/session/verification/verification_20260629_053610_aa590b/verification.json`
- `npm run check`
  - Codexus 증적:
    `.codexus/session/verification/verification_20260629_054145_b74456/verification.json`
- `npm run docs:check`
  - 결과: markdown 215개 확인, 깨진 로컬 링크 없음.
- `npm run repo:hygiene`
  - 결과: tracked file 221개 확인.
- `cx release policy --gate --json`
  - 결과: 영어/한국어 릴리즈 정책 문서 추가 후 통과.

## 게시 후 검증

`v0.1.3` 태그 푸시 후 다음을 확인합니다.

```powershell
gh release view v0.1.3 --repo raeseoklee/agentmux
gh attestation verify .\AgentMux_0.1.3_x64-setup.exe --repo raeseoklee/agentmux --signer-workflow raeseoklee/agentmux/.github/workflows/release.yml
Get-FileHash -Algorithm SHA256 .\AgentMux_0.1.3_x64-setup.exe
Get-Content .\AgentMux_0.1.3_x64-setup.exe.sha256
```

GitHub 릴리즈 노트에는 최종 설치 파일 이름과 SHA256이 포함되어야 합니다.

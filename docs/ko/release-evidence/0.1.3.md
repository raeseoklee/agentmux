# AgentMux 0.1.3 릴리즈 증적

일자: 2026-06-29

## 범위

AgentMux 0.1.3은 공개 릴리즈 준비에 초점을 둔 Windows-only 패치
릴리즈입니다.

- Node, Tauri, Rust workspace, lock 파일의 버전을 0.1.2에서 0.1.3으로
  올렸습니다.
- 공개 README, 보안 정책, 이슈 템플릿, PR 템플릿, 알려진 제한사항 문서를
  정리했습니다.
- 영어와 한국어 릴리즈 정책 문서를 추가했습니다.
- GitHub release workflow의 publish 경로에서 사용하는 action을 commit SHA로
  고정했습니다.
- 단일 pane 터미널 스크롤 레이아웃을 수정하고, split이 아닌 xterm pane에서
  휠 스크롤이 동작하는지 확인하는 Playwright 회귀 테스트를 추가했습니다.
- 레거시 cmux 프로젝트 설정이 실제로 존재하지 않는 경우 설정 진단에서
  비활성 `cmux_project` 항목을 숨기도록 정리했습니다.

## 게시 전 검증

태그를 푸시하기 전에 다음 로컬 검증을 완료했습니다.

- `npm run version:check -- --tag v0.1.3`
  - Codexus 증적:
    `.codexus/session/verification/verification_20260629_061501_0ada53/verification.json`
- `npm --prefix apps/desktop run build`
  - Codexus 증적:
    `.codexus/session/verification/verification_20260629_061533_bfe4d1/verification.json`
- `npm run check`
  - 결과: 로컬 통과. Codexus의 build/check 결합 실행은 build 통과 후 기본
    명령 타임아웃에 걸렸기 때문에, `check`는 더 긴 셸 타임아웃으로 직접
    재실행해 통과를 확인했습니다.
- `npx playwright test --grep "single pane terminal keeps"`
  - 결과: 로컬 통과. scrollback이 생성된 단일 xterm pane에서 휠 스크롤이
    동작함을 확인했습니다.
- `npx playwright test --grep "settings hides cmux project diagnostics|settings can migrate preview cmux project config"`
  - 결과: 로컬 통과. 기본 설정 화면에는 비활성 `cmux_project` 진단이
    노출되지 않고, 레거시 마이그레이션은 계속 동작함을 확인했습니다.
- `rustup run stable-x86_64-pc-windows-msvc cargo test desktop_config_diagnostics -- --nocapture`
  - 결과: 로컬 통과. Tauri native 설정 진단에서도 존재하지 않는 레거시
    cmux 프로젝트 소스가 숨겨짐을 확인했습니다.
- `npm run docs:check`
  - 결과: markdown 216개 확인, 깨진 로컬 링크 없음.
  - Codexus 증적:
    `.codexus/session/verification/verification_20260629_061501_0ada53/verification.json`
- `npm run repo:hygiene`
  - 결과: tracked file 221개 확인.
  - Codexus 증적:
    `.codexus/session/verification/verification_20260629_061501_0ada53/verification.json`
- `cx release policy --gate --json`
  - 결과: 영어/한국어 릴리즈 정책 문서 추가 후 통과.

## 게시 후 검증

`v0.1.3` 태그 푸시 후 다음을 확인합니다.

```powershell
gh release view v0.1.3 --repo raeseoklee/agentmux
gh attestation verify .\AgentMux_0.1.3_x64-setup.exe --repo raeseoklee/agentmux --signer-workflow raeseoklee/agentmux/.github/workflows/release.yml
Get-FileHash -Algorithm SHA256 .\AgentMux_0.1.3_x64-setup.exe
Get-Content .\AgentMux_0.1.3_x64-setup.exe.sha256
```

GitHub 릴리즈 노트에는 최종 설치 파일 이름과 SHA256이 포함되어야 합니다.

# AgentMux 0.1.3 Release Evidence

Date: 2026-06-29

## Scope

AgentMux 0.1.3 is a Windows-only patch release focused on public release
readiness:

- Version bump from 0.1.2 to 0.1.3 across Node, Tauri, Rust workspace, and lock
  files.
- Public-facing README, security policy, issue templates, PR template, and
  known-limitations documentation.
- Release policy documentation in English and Korean.
- GitHub release workflow hardening by pinning publish-path actions to commit
  SHAs.
- Single-pane terminal scroll layout fix, plus a Playwright regression test that
  verifies wheel scrolling inside a non-split xterm pane.

## Post-Publish Verification

After pushing `v0.1.3`, verify:

```powershell
gh release view v0.1.3 --repo raeseoklee/agentmux
gh attestation verify .\AgentMux_0.1.3_x64-setup.exe --repo raeseoklee/agentmux --signer-workflow raeseoklee/agentmux/.github/workflows/release.yml
Get-FileHash -Algorithm SHA256 .\AgentMux_0.1.3_x64-setup.exe
Get-Content .\AgentMux_0.1.3_x64-setup.exe.sha256
```

The GitHub release notes should contain the final installer name and SHA256.
